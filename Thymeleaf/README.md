# Thymeleaf Dashboard (Minimal Spring Boot example)

This is a minimal Spring Boot application that demonstrates a simple dashboard rendered with Thymeleaf.

Requirements
- Java 17+
- Maven 3.6+

Run locally (PowerShell)

Install dependencies and build jar:

```powershell
mvn -DskipTests package
```

Run directly with Spring Boot plugin:

```powershell
mvn spring-boot:run
```

Or run the packaged jar:

```powershell
java -jar target/thymeleaf-dashboard-0.0.1-SNAPSHOT.jar
```

Open http://localhost:8080/ in your browser.
